A free, customizable foundation for building engaging clicker games like "Cookie Clicker" or "Cat Pop". Perfect for beginners and experienced developers looking to create addictive incremental games.


## ‚ú® Features

- **Simple Core Mechanics** - Easy-to-understand clicking gameplay
- **Automatic Saving** - Player progress saved via browser cookies
- **Customizable UI** - Easily modify visuals to match your theme
- **Leaderboard Integration** - Built-in competitive elements
- **Mobile-Friendly** - Responsive design works on all devices
- **Lightweight** - Fast loading with minimal dependencies

## üöÄ Getting Started

### Prerequisites

- Basic knowledge of HTML, CSS, and JavaScript
- A code editor (VS Code, Sublime Text, etc.)
- A modern web browser

### Installation

1. **Download** the latest release from our [GitHub Releases](https://github.com/awiones/Clicker-Game-Template/releases) page
2. **Extract** the ZIP file to your preferred location
3. **Open** the project in your favorite code editor
4. **Launch** `index.html` in your browser to test the base game

### Customization

Easily make the game your own by modifying these key files:

- **`index.html`** - Adjust the game structure and layout
- **`styles.css`** - Customize colors, fonts, and visual elements
- **`main.js`** - Enhance gameplay mechanics and add new features

#### Quick Customization Tips

```javascript
// In main.js - Change point value per click
const POINTS_PER_CLICK = 5; // Default: 1

// In styles.css - Change the main button color
.click-button {
  background-color: #ff6b6b; /* Change to your preferred color */
}
```

### Deployment

Host your finished game on any web server or free platforms like:

- **GitHub Pages** - Perfect for open-source projects
- **Netlify** - Offers continuous deployment from Git
- **Vercel** - Great for frontend projects with simple setup

## üéÆ How to Play

1. **Click the button** as fast as possible to earn points
2. **Unlock upgrades** by spending your earned points
3. **Progress saves automatically** using cookies
4. **Compete on the leaderboard** against other players
